# Oculus

> WoW Midnight (12.0) PvP Addon Suite

**Oculus** (라틴어: 눈)는 PvP 전투에서 필요한 모든 정보를 한눈에 볼 수 있게 해주는 애드온 스위트입니다.

## 모듈

| 모듈 | 설명 |
|------|------|
| **Oculus** | 코어 (모듈 관리, 공통 설정) |
| **Oculus_UnitFrames** | Player/Target/Focus 버프/디버프 필터 |
| **Oculus_RaidFrames** | 파티 버프/디버프 + 쿨다운 트래킹 + 시전 알림 |
| **Oculus_ArenaFrames** | 아레나 프레임 정렬 + 버프/디버프 필터 |

## 설치

1. `Oculus` 폴더들을 `World of Warcraft/_retail_/Interface/AddOns/`에 복사
2. 게임 실행 후 `/oculus` 입력

## 명령어

```
/oculus              - 설정 열기
/oculus test         - 테스트 모드
/oculus test <module> - 특정 모듈 테스트
```

## 요구사항

- World of Warcraft: Midnight (12.0.0+)
